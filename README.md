# Tailwind Nested

With Tailwind Nested you can turn your long Tailwind classes into hierarchical view, make changes and compose it back.

## Features

![Demo](./images/demo.gif)

## Keybindings

- <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>e</kbd> - Toggle nested view
